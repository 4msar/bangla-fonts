![BenSen](/images/fonts/bensen.jpg)

Developer: [Subrata Sen](http://www.subratasen.com/)

### Embed Font

To embed your selected fonts into a webpage, copy this code into the <head> of your HTML document.

##### STANDARD

```css
<link href="https://cdn.msar.me/fonts/bensen/font.css" rel="stylesheet">
```

##### Or, IMPORT in CSS

```css
<style>
@import url('https://cdn.msar.me/fonts/bensen/font.css');
</style>
```

### Specify in CSS

Use the following CSS rules to specify these families:

```css
font-family: 'BenSen', sans-serif;
```

#### Example

```css
body {
    font-family: 'BenSen', Arial, sans-serif !important;
}
```
